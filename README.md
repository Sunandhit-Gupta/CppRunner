# 🚀 C++ Online Compiler (Next.js)

A modern, responsive web-based C++ compiler built using **Next.js** that allows users to:

- ✍️ Write C++ code
- ⌨️ Provide custom input
- 📂 Upload `.txt` input files
- ⚡ Compile and run code instantly
- 📤 View output in real-time

---

## 🌟 Features

- 🧠 **C++ Code Execution** using `g++`
- 📥 **Custom Input Support**
- 📂 **.txt File Upload for Input**
- 🔄 **Loader while execution**
- 🖥️ **Responsive UI (Mobile + Desktop)**
- ⚡ **Fast Execution with Timeout Handling**
- 🎯 **Codeforces-style multiple test cases support**

---

## 🖼️ UI Highlights

- Split layout:
  - Left → Code Editor
  - Right → Input & Output
- Clean and minimal design
- No unnecessary scrolling
- Real-time feedback with loader

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, Tailwind CSS
- **Backend:** Node.js (Next.js API Routes)
- **Compiler:** `g++` (GCC)
